#Similarity: 
Both type of exception can be catched and save program if user wants
Both type of exception can be throws from method declaration
Both type of exception can be throw from inside method

#Differences:
1. Checked has to be handled in some way
   Unchecked need not be handled in any way

2. Checked can propogates to caller method
   Unchecked will not propogates to caller method

3. Checked can be converted into unchecked if user dont want to save the code
